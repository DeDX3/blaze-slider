# `loop`

A boolean value indicating whether the slider should have an infinite loop or not

## default

```typescript
loop: true
```

<br />

### Example : `loop: true` (default)

```javascript
new BlazeSlider(el, {
  all: {
    loop: true
    slidesToShow: 3,
  },
})
```

<ExampleSlider
  addNav={true}
  config={{
    all: {
      loop: true,
      slidesToShow: 3,
    },
  }}
>
  <div> 1 </div>
  <div> 2 </div>
  <div> 3 </div>
  <div> 4 </div>
  <div> 5 </div>
  <div> 6 </div>
</ExampleSlider>
