# `enableAutoplay`

import { ExampleSlider } from '@site/src/components/ExampleSlider'

it is a boolean value indicating whether autoplay should be enabled or not.

## default

```typescript
enableAutoplay: false
```

<br />

### Example `enableAutoplay: true`

In this example, pagination and navigation buttons are also added, so that you can see the impact of auotplay on them.

`stopAutoplayOnInteraction` is also turned off on this example (which is enabled by default) to keep the autoplay enabled.

```javascript
new BlazeSlider(el, {
  all: {
    enableAutoplay: true,
    slidesToShow: 3,
    // keep the autoplay on even if user interacts with it
    stopAutoplayOnInteraction: false,
  },
})
```
