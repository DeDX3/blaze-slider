# `autoplayDirection`

import { ExampleSlider } from '@site/src/components/ExampleSlider'

It indicates the direction of the autoplay - `to right` or `to left`

## possible values

```typescript
autoplayDirection: 'to left' | 'to right'
```

## default

```typescript
autoplayDirection: 'to left'
```

<br />

### Example `autoplayDirection: "to left"` (default)

```javascript
new BlazeSlider(el, {
  all: {
    autoplayDirection: 'to left', // <--
    enableAutoplay: true,
    slidesToShow: 3,
    enableAutoplay: true,
  },
})
```

<ExampleSlider

  }}
>
