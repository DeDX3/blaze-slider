# `slideGap`

length of gap between the slides using any absolute or relative [css length units](https://developer.mozilla.org/en-US/docs/Learn/CSS/Building_blocks/Values_and_units#lengths)

## default

```typescript
slideGap: '20px'
```

<br />

blaze-slider will set the value of `slideGap` as value of `--slide-gap` css variable on the target `.blaze-slider` element

:::note
Remember to also configure the `--slide-gap` in your own css file as well to avoid layout shifts as explained in [this tutorial](/docs/Tutorial/layout-shift)
:::

<br />

### Example `slideGap: '50px'`

```javascript
new BlazeSlider(el, {
  all: {
    slideGap: '50px',
    slidesToShow: 3,
  },
})
```

<ExampleSlider
  config={{
    all: {
      slideGap: '50px',
      slidesToShow: 3,
    },
  }}
>

  <div> 2 </div>
  <div> 3 </div>
  <div> 4 </div>
  <div> 5 </div>
  <div> 6 </div>
</ExampleSlider>
