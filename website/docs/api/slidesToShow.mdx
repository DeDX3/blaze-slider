# `slidesToShow`

The number of slides to show in the slider at a time.

## default

```typescript
slidesToShow: 1
```

<br />

blaze-slider will set the value of `slidesToShow` as value of `--slides-to-show` css variable on the target `.blaze-slider` element

:::note
Remember to also configure the `--slides-to-show` in your CSS file to avoid layout shifts as explained in [this tutorial](/docs/Tutorial/layout-shift)
:::

<br />

### Example

```javascript
new BlazeSlider(el, {
  all: {
    slidesToShow: 3,
  },
})
```
