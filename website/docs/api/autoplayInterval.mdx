# `autoplayInterval`

import { ExampleSlider } from '@site/src/components/ExampleSlider'

it indicates the interval of time in milliseconds between autoplay transition (scroll).

For Example, if `autoplayInterval: 3000` is configured, the blaze-slider will scroll the slider every 3 seconds. (if autoplay is enabled)

## default

```typescript
autoplayInterval: 3000
```

<br />

### Example `autoplayInterval: 3000` (default)

```javascript
new BlazeSlider(el, {
  all: {
    enableAutoplay: true,
    autoplayInterval: 3000,
    slidesToShow: 3,
  },
})
```
