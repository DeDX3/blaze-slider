# `transitionTimingFunction`

import { ExampleSlider } from '@site/src/components/ExampleSlider'

The CSS [transition-timing-function](https://developer.mozilla.org/en-US/docs/Web/CSS/transition-timing-function)
to be used on the slide items, such as `ease`, `ease-in`, `east-out`, `ease-in-out`, `linear` or custom cubic-bezier curve

## default

```typescript
transitionTimingFunction: 'ease'
```

<br />

### Example `transitionTimingFunction: "ease"` (default)

```javascript
new BlazeSlider(el, {
  all: {
    enableAutoplay: true,
    transitionTimingFunction: 'ease', // (default)
    slidesToShow: 3,
  },
})
```

<ExampleSlider
  addNav={true}
  config={{
    all: {
      enableAutoplay: true,
      transitionTimingFunction: 'ease', // (default)
      slidesToShow: 3,

### Example custom cubic-bezier curve

```javascript

  all: {
    enableAutoplay: true,
    transitionTimingFunction: 'cubic-bezier(.18,.74,.2,1.27)',
    slidesToShow: 3,
  },
})
```
