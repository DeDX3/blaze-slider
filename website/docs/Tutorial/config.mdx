# Provide Config

Refer to [BlazeConfig](/docs/api/BlazeConfig) to learn about the structure of BlazeConfig, all the different properties that can be configured and how the configuration cascading works

## Example

```javascript
new BlazeSlider(el, {
  all: {
    enableAutoplay: true,
    autoplayInterval: 2000,
    transitionDuration: 300,
    slidesToShow: 3,
  },
  '(max-width: 900px)': {
    slidesToShow: 2,
  },
  '(max-width: 500px)': {
    slidesToShow: 1,
  },
})
```
