# Slider Benchmark

<br/>

## Render 1 slider with 10 slides

Benchmark compares the time taken by each library to render 1 slider with 10 slides with same config and same style

| Library      | Time   |
| ------------ | ------ |
| blaze-slider | 3.2ms  |
| swiper       | 40.7ms |
| slick        | 59.7ms |
